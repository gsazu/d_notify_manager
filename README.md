# D Notify Manager
`A App that can filter text from notificaitions and play a different sound.`


## Steps
1. Install the app.
2. enable notification access permission to this app.
3. add the text by click on the textfield.
4. select the tune for that text on click of select tune button and then click add button.
5. you can delete the text from list after anytime.
6. open this app info by long click on app and then in the battery section, select the unrestricted option.
7. now you can close the app completely,
8. check via sending the same text from the another phone to this phone to get this phone a notification.
9. the notification appear for that text and then you ca stop the tune on stop button of the notification.
   ### Note : the latter's in this app that are checked are not case-sensitive to make the better performance.
