# D Notify Manager
###`A App that can filter text from otificaitions and play a different sound.`


## Steps
1. Install the app.
2. enable notification access permission to this app.
3. add the text by click on add button,
4. you can delete the text from list after anytime.
5. open this app info by long click on app and then in the battery section, select the unrestricted option.
6. now you can close the app completely,
7. check via sending the same text from the another phone to this phone to get this phone a notification.
8. raw folder sound will perform.
   ### Note : the latters in this app that are checked are not case sensitive to make the better permformance.
